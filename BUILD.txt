You need Apache Maven to build.

See: http://maven.apache.org/

Available goals:

  1. install - Generates binary files in ${basedir}/target.
  2. jar - Generates ${basedir}/target/junique-${version}.jar file.
  3. javadoc - Generates javadoc api documentation in ${basedir}/target/site/apidocs.

General build properties can be managed within the pom.xml file.
